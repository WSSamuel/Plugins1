== Frequently Asked Questions ==

= 1. How to use Twitter feed widget pro plugin. =

Download Custom twitter widget pro plugin from WordPress repository and extract it.
Go to WordPres Admin dashboard activate plugin from plugin dashboard.
Go to widget area of WordPress dashboard and set your twitter username and enjoy it
