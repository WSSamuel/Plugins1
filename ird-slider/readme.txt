Ird Slider, crea galerías de imágenes con descripciones y enlaces. 

== Description ==

Ird Slider permite crear múltiples sliders para tu web. Es sencillo de utiliizar, solamente debes rellenar la información como si fuera un post de tu blog y completar algún dato de configuración adicional (pero sólo si quieres).
No necesitas configurarlo, solo accedes a la configuración para obtener el shortcode.
No necesitas traducirlo, solamente muestra tus textos.

Una vez instalado el plugin, crea una nueva entrada de Ird Slider desde el menú.
Rellena el título y el contenido a mostrar, una imagen (destacada), un tipo (crea tantos como quieras solamente sirve para que puedas mostrar diferentes sliders, es decir, es una agrupación) y, si quieres, orden y url que abrirá al hacer click.
Desde la configuración del plugin, indica los valores para presentarlo a tu gusto. Presta especial atención al tipo, que tiene que ser el mismo que se ha indicado para las imágenes. Tras pulsar el botón, se generará el shortcode listo para copiar y pegar en cualquier página o entrada.

== Screenshots ==

1. Una vez instalado el plugin, crea una nueva entrada de Ird Slider desde el menú.
2. Rellena el título y el contenido a mostrar junto a cada imagen.
3. Elige la imagen de tu biblioteca de medio o sube una nueva.
4. El campo etiquetas, es opcional. Se mostrará debajo del título. Si indicas una url, al pinchar en la imagen o texto, se abrirá en el lugar indicado en la configuración del slider. Cada tipo es un slider diferente, debe ser igual en todos los casos y en la configuración. El orden admite duplicidades y no tiene que ser consecutivo.
5. Desde la configuración del plugin, indica los valores para presentarlo a tu gusto. Presta especial atención al tipo, que tiene que ser el mismo que se ha indicado para las imágenes. Tras pulsar el botón, se generará el shortcode listo para copiar y pegar en cualquier página o entrada.
