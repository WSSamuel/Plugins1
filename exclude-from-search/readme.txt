Exclude Page and Attachment post types from search results.

== Description ==

Exclude Page and Attachment post types from search results.

* Very fast execution
* No configuration required

== Installation ==

1. Unzip the exclude-from-search.zip folder.
1. Upload the exclude-from-search folder to your /wp-content/plugins directory.
1. In your WordPress dashboard, head over to the Plugins section.
1. Activate Exclude From Search.
