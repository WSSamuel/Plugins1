== Description ==

This plugin will automaticaly add a smoothly scrolling feature on your website with your mouse and keyboard.

== Installation ==

= For automatic installation: =
* The simplest way to install is to click on 'Plugins' then 'Add New' and type 'Easy Smooth Scroll' in the search field.
* Click 'Install Now' and have Fun!

= For manual installation: =
* Login to your website and go to the Plugins section of your admin panel.
* Click the Add New button.
* Under Install Plugins, click the Upload link.
* Select the plugin zip file (easy-smooth-scroll.zip) from your computer then click the Install Now button.
* You should see a message stating that the plugin was installed successfully.
* Click the Activate Plugin link.


= Installation from server: =
* You should have access to the server where WordPress is installed. If you don't, see your system administrator.
* Copy the plugin zip file (easy-smooth-scroll.zip) up to your server and unzip it somewhere on the file system.
* Copy the "easy-smooth-scroll" folder into the /wp-content/plugins directory of your WordPress installation.
* Login to your website and go to the Plugins section of your admin panel.
* Look for "Easy Smooth Scroll" and click Activate.
