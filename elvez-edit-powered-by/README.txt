== Description ==

This plugin can edit the 'Powered by' text in footer of the twentytwenty Wordpress theme.

標準テーマの"Powerd by Wordpress"表記を、非表示または上書きできるプラグインです。

A few notes about the sections above:

* Replace "Powered by Wordpress" text to custom text and link url.
* Hide "Powered by Wordpress" text.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/elvez-edit-powered-by` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Dashboard->Elvez Edit Powered By screen to configure the plugin
