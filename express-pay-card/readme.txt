== Описание ==

«Экспресс Платежи: Интернет-Эквайринг» для WooCommerce, 
плагин для простого подключения приема платежей банковскими платежными 
картами (Visa, MasterCard, Maestro, БЕЛКАРТ), картами рассрочек (включая карту Халва).

Подключение к нашему сервису «Экспресс Платежи» позволит расширить возможности оплаты товара или услуг. 
Ваши покупатели смогут осуществлять платежи прямо на сайте, а также через интернет-банкинг, SMS-банкинг, мобильный банкинг, 
во всех пунктах банковского обслуживания, почтовых отделениях. 


== Сервис «Экспресс Платежи» предоставляет следующие возможности:  ==

    * Функциональный личный кабинет; 
    * Прием платежей в ЕРИП; 
    * Прием платежей по картам рассрочек (включая карту Халва);
    * Прием платежей «онлайн» через банковские платежные карты (Visa, MasterCard, Maestro, БЕЛКАРТ); 
    * Прием платежей по QR-коду ; 
    * Одновременная работа с ЕРИП, E-POS и банковскими платежными картами; 
    * Совместный доступ любого количества сотрудников в личном кабинете; 
    * Разделение прав доступа для разных категорий сотрудников; 
    * Мобильное приложение; 
    * Наличие API для интеграции с любым сторонним программным обеспечением; 
    * Мгновенное выставление счетов и принятие платежей ЕРИП (E-POS) в режиме «онлайн»; 
    * Ручное выставление счетов на оплату через личный кабинет; 
    * Прием произвольных платежей без выставления счета; 
    * Групповое выставление счетов, 
    * Периодическое выставление счетов; 
    * Постоянный счет многократно доступный к оплате; 
    * Публичный счет в виде ссылки, которую можно отправить по электронной почте, в SMS-сообщении, с помощью мессенджеров или в социальных сетях; 
    * Импорт / экспорт счетов; 
    * Уведомления клиентам о выставлении счета по электронной почте; 
    * Бесплатные уведомления о платежах (SMS, Email, в браузер); 
    * Неограниченный срок хранения реестров о проведенных операциях; 
    * Выписки по полученным платежам для ведения отчетности; 
    * И многое другое.

= Демонстрационная версия личного кабинета =

На нашем сайте Вам доступна демонстрационная версия личного кабинета («Демо вход»), где Вы можете детально ознакомиться с возможностями личного кабинета. Демо-версия личного кабинета доступна по адресу [Демо-кабинет](https://demo.express-pay.by)

== О компании ==

ООО «ТриИнком» является официальным Агрегатором системы ЕРИП (АИС
«Расчет») и резидентом ПВТ (Парка высоких технологий), в рамках сервиса
«Экспресс Платежи» оказывает услуги по приему платежей в системе ЕРИП
(E-POS) и по банковским платежным картам «онлайн». Подробная информация о
сервисе представлена на сайте [express-pay.by](https://express-pay.by)
Наш сервис принимает участие в престижных конкурсах, конференциях и
ООО «ТриИнком» - О подключении к системе АИС «Расчет» (ЕРИП) и банковским платежным картам в режиме «онлайн» рейтингах Беларуси, на которых представляет разработанные и внедренные проекты для комфортной и слаженной работы с платежами. Ежегодно возможности и функциональность сервиса отмечают призовыми местами, с которыми можно ознакомится по адресу [Награды](https://express-pay.by/nagrady)
