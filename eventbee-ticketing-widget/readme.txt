== Description ==
Eventbee Ticket Widget plugin is a great solution for Event Managers who wish to sell their tickets having Ticket Widget on their WordPress sites.
Eventbee Ticket Widget appropriately blends with any type of themes and the widget can be placed anywhere according to the user's wish.
Using this widget an attendee can purchase tickets for the event by following certain process of selecting a ticket, filling up the profiles, selecting payment method to pay the amount and finally a confirmation message will be displayed once the payment is completed successfully.

== Arbitrary section ==
Not just selling tickets: 

* Manager can provide constraints to the tickets by using Conditional ticketing.
* Manager can control who to register for the event.
* Manager can ask questions apart from default one.
* Manager can add discounts for the tickets.
* Manager can customize the look and feel of the widget according to the site.

== A brief Markdown Example ==
If your event ID is 1234567890, in the page where you want to place the widget, just put
[eventbeeticketwidget eid="1234567890"]
It will display widget with a default width of 700px

To display with a different width (say 800px), use
[eventbeeticketwidget eid="1234567890" width="800"]

Or you may visit your event manage console in [eventbee.com](https://www.eventbee.com), go to Integration and get wordpress widget code.
