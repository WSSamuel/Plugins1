Galaksion tag manager is a simple tool for placing Galaksion ad network's ad tags to your WordPress website.

== Description ==

Plugin allows to put ad tags at the different places of your WordPress site pages.

The user interface is pretty simple:

1. Install the plugin
2. Go to the "Plugins" page
3. Find "Galaksion tag manager" and click the "Scripts" link
4. Choose the placement and push the "Add code" button
5. Then paste your ad tag and push "Add" button
6. That's all -- ad tag will be working at the place you've chosen.

== Installation ==

#### Installing from the WordPress control panel

1. Go to the page "Plugins > Add New".
1. Input the name "Galaksion Tag Manager" in the search field
1. Find the "Galaksion Tag Manager" plugin in the search result and click on the "Install Now" button, the installation process of plugin will begin.
1. Click "Activate" when the installation is complete.

#### Installing with the archive

1. Go to the page "Plugins > Add New" on the WordPress control panel
1. Click on the "Upload Plugin" button, the form to upload the archive will be opened.
1. Select the archive with the plugin and click "Install Now".
1. Click on the "Activate Plugin" button when the installation is complete.

#### Manual installation

1. Upload the folder `galaksion-tag-manager` to a directory with the plugin, usually it is `/wp-content/plugins/`.
1. Go to the page "Plugins > Add New" on the WordPress control panel
1. Find "Galaksion Tag Manager" in the plugins list and click "Activate".

### After activation

After the plugin is successfully installed the menu item "Scripts" will appear in the menu "Settings" of the WordPress control panel.
